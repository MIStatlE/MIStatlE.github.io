# Data Structures & Algorithms

This chapter introduces common data structures and classic algorithms, forming the core of programming skills and computer science learning.

## Data Structures

- **Linear structures**: arrays, linked lists, stacks, queues, etc.;
- **Trees**: binary trees, binary search trees, heaps, B‑trees, etc.;
- **Graphs**: adjacency lists and matrices;
- **Hash tables**: hash functions and collision resolution strategies.

Understand the appropriate scenarios, time complexity and space complexity of each structure.

## Classic Algorithms

- **Sorting algorithms**: bubble sort, quick sort, merge sort, heap sort, etc.;
- **Searching algorithms**: linear search, binary search, hashing-based search;
- **Graph algorithms**: Dijkstra’s shortest path, Prim and Kruskal’s minimum spanning tree, topological sort, etc.;
- **Dynamic programming**: Fibonacci numbers, knapsack problem, longest common subsequence and more.

When learning algorithms, first grasp the underlying ideas, then reinforce them through manual derivation and code implementation.

## Analysis Methods

- **Time and space complexity**: expressed using big‑O notation;
- **Asymptotic analysis**: compare the efficiency of different algorithms;
- **Practice & optimization**: identify bottlenecks in practical applications and optimize performance.

Organizing solved problems and summarizing common approaches helps prepare for interviews and competitions.
