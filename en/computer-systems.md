# Computer Systems

Understanding how computer systems work helps you grasp how programs run. This chapter introduces computer organization, operating systems, networks and databases.

## Computer Architecture

- **Hardware components**: the structure and interaction of the CPU, memory, buses and I/O devices;
- **Instruction sets**: machine instructions and assembly language;
- **Pipelining and parallelism**: techniques to improve processor performance.

## Operating Systems

- **Processes and threads**: scheduling policies, synchronization and mutual exclusion;
- **Memory management**: segmentation, paging and virtual memory;
- **File systems and I/O**: data storage and input/output management.

## Computer Networking

- **Layered models**: application, transport, network, data link and physical layers;
- **Protocols**: TCP/IP, HTTP/HTTPS, DNS and more;
- **Network security**: basic encryption and secure communication concepts.

## Database Basics

- **Relational databases**: tables, primary keys, foreign keys and SQL queries;
- **Transactions and concurrency control**: ACID properties;
- **NoSQL**: key‑value stores, document databases and more.

Mastering these system‑level topics helps you build more stable and efficient software.
