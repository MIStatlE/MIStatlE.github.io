# Essential Tools

During self‑study, mastering certain tools can greatly enhance your efficiency. This chapter introduces common tools and usage suggestions.

## Python and Jupyter Notebook

Python is the main language for experiments in algorithms and machine learning. Installing Anaconda or Miniconda is recommended, since it simplifies dependency management. Jupyter Notebook is ideal for interactive programming and documentation, allowing you to combine code, charts and narrative text.

### Installation and usage

1. Install [Python](https://www.python.org/) and [Jupyter Notebook](https://jupyter.org/) (use `pip` or `conda`).
2. Start the interactive environment with `jupyter notebook`.
3. Learn common libraries such as NumPy, Pandas, Matplotlib and Scikit‑Learn for scientific computing and data analysis.

## Git and Version Control

Git is used to manage versions of code and documents, facilitating collaboration and history tracking. Install a Git client and learn these common commands:

- `git clone`: clone a remote repository;
- `git pull`: fetch the latest changes;
- `git add`, `git commit`: stage and commit local changes;
- `git push`: push commits to a remote repository.

Understanding branching and merging is also valuable; refer to the book *Pro Git* for deeper study.

## Command‑Line Tools

Becoming proficient with the command line can dramatically improve productivity. Suggested topics:

- Basic shell commands: `ls`, `cd`, `cp`, `mv`, `rm`, etc.;
- Text processing tools: `grep`, `sed`, `awk`;
- Terminal enhancement: `tmux`, `zsh`, `oh‑my‑zsh`, etc.

## Other Tools

Depending on your learning direction, you may also use:

- **Math and scientific computing**: MATLAB, Octave, SageMath;
- **Machine learning libraries**: TensorFlow, PyTorch, Keras;
- **Visualization**: Seaborn, Plotly, Bokeh;
- **Editors and IDEs**: Visual Studio Code, PyCharm, Vim.

You can expand this list as needed and explore more tools in depth.
