# 数据结构与算法

本章介绍常见数据结构与经典算法，是编程能力和计算机科学学习的核心部分。

## 数据结构

- **线性结构**：数组、链表、栈、队列等；
- **树结构**：二叉树、二叉搜索树、堆、B 树等；
- **图结构**：邻接表与邻接矩阵表示；
- **散列表**：哈希函数与碰撞解决方法。

了解每种结构的适用场景、时间复杂度和空间复杂度。

## 经典算法

- **排序算法**：冒泡排序、快速排序、归并排序、堆排序等；
- **查找算法**：顺序查找、二分查找、哈希查找；
- **图论算法**：Dijkstra 最短路径、Prim 与 Kruskal 最小生成树、拓扑排序等；
- **动态规划**：斐波那契数列、背包问题、最长公共子序列等。

学习算法时，建议先理解思想，再通过手工推演和代码实现巩固。

## 分析方法

- **时间复杂度与空间复杂度**：通过大 O 符号表征；
- **渐进分析**：比较不同算法的效率；
- **实践与优化**：在实际应用中分析瓶颈并优化性能。

整理做过的习题和经验，总结常见思路，有助于应对算法面试和竞赛。
