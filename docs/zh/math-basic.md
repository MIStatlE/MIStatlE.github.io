# 数学基础

本章记录数学基础知识，包括微积分、线性代数和概率统计等。建议按主题逐步学习，每个小节以二级标题开始。

## 微积分

- **主要概念**：极限、导数、积分等。
- **学习要点**：掌握求导和积分的基本技巧以及相关定理，例如泰勒展开和积分换元法。

## 线性代数

- **主要概念**：向量与矩阵、线性变换、特征值与特征向量、矩阵分解等。
- **学习要点**：理解矩阵运算及其几何意义，掌握常用分解方法，如特征分解、SVD。

## 概率与统计

- **主要概念**：随机变量、概率分布、期望与方差、假设检验等。
- **学习要点**：熟悉常见分布及其性质，能够应用基础统计方法分析数据。

更多高级主题，如优化方法、数值计算等，可在后续章节中深入学习。
