# 计算机系统基础

掌握计算机系统的工作原理有助于理解程序的运行机制。本章将介绍计算机组成、操作系统、网络与数据库等基础内容。

## 计算机组成原理

- **硬件架构**：CPU、内存、总线、I/O 设备的结构与交互；
- **指令集**：机器指令与汇编语言；
- **流水线与并行处理**：提高处理器性能的技术。

## 操作系统

- **进程与线程**：调度策略、同步互斥；
- **内存管理**：分段、分页、虚拟内存；
- **文件系统与 I/O**：数据存储与输入输出管理。

## 计算机网络

- **分层模型**：应用层、传输层、网络层、数据链路层和物理层；
- **协议**：TCP/IP、HTTP/HTTPS、DNS 等；
- **网络安全**：基本的加密与安全通信概念。

## 数据库基础

- **关系型数据库**：表、主键、外键与 SQL 查询；
- **事务与并发控制**：ACID 性质；
- **NoSQL 数据库**：键值存储、文档数据库等。

综合掌握这些系统层面的知识将帮助你构建更稳定高效的软件系统。
