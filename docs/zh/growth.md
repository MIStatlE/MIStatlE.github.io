# 心智成长

自学不仅是知识积累的过程，也是自我认知和心智成长的过程。本章分享一些心理学书籍、博客和实践建议，帮助你保持积极心态、提升学习效率。

## 推荐书籍

- **《终身成长》Carol S. Dweck**：阐述了成长型思维模式的重要性；
- **《刻意练习》Anders Ericsson**：讲述如何通过有针对性的练习提高技能；
- **《原子习惯》James Clear**：介绍建立好习惯与戒除坏习惯的方法；
- **《高效能人士的七个习惯》Stephen R. Covey**：关于个人管理与领导力的经典著作。

## 优秀博客

- **Wait But Why**：以深入浅出的方式讨论复杂话题；
- **LessWrong**：聚焦理性思考与决策科学；
- **Overreacted**：由 React 核心成员 Dan Abramov 撰写，涉及编程和成长经验。

## 心理调节与效率

- 保持规律作息和适度运动，保证精力充沛；
- 设定可实现的目标，并及时给予自己积极反馈；
- 练习正念冥想或写日记，提高专注力与情绪管理。

不断提升自我认知和心理素质，有助于长期坚持学习并保持良好状态。
