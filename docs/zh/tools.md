# 必学工具

在自学过程中，熟练使用一些工具可以大幅提升效率。本章整理常见工具的介绍和使用建议。

## Python 与 Jupyter Notebook

Python 是进行算法与机器学习实验的主要语言。推荐安装 Anaconda 或 Miniconda，方便管理依赖。Jupyter Notebook 适合交互式编程与文档编写，可同时包含代码、图表和文字说明。

### 安装与使用

1. 安装 [Python](https://www.python.org/) 和 [Jupyter Notebook](https://jupyter.org/)（推荐使用 `pip` 或 `conda`）。
2. 使用 `jupyter notebook` 命令启动交互式环境。
3. 学习常用库，如 NumPy、Pandas、Matplotlib、Scikit‑Learn 等，用于科学计算与数据分析。

## Git 与版本控制

Git 用于管理代码和文档的版本，方便协作和历史追踪。推荐安装 Git 客户端并学习如下常用命令：

- `git clone`：克隆远程仓库；
- `git pull`：拉取最新修改；
- `git add`、`git commit`：提交本地修改；
- `git push`：推送到远程仓库。

了解基本的分支管理和合并操作，可查阅《Pro Git》等书籍深入学习。

## 命令行工具

熟练使用命令行可以极大提高操作效率。建议学习以下内容：

- 基本 Shell 命令：如 `ls`、`cd`、`cp`、`mv`、`rm` 等；
- 文本处理工具：如 `grep`、`sed`、`awk`；
- 终端增强工具：如 `tmux`、`zsh`、`oh‑my‑zsh` 等。

## 其他工具

根据学习方向，还可能用到以下工具：

- 数学与科学计算：MATLAB、Octave、SageMath；
- 机器学习库：TensorFlow、PyTorch、Keras；
- 可视化与绘图：Seaborn、Plotly、Bokeh；
- 代码编辑器与 IDE：Visual Studio Code、PyCharm、Vim。

你可以根据需要不断扩充工具清单，学习更多细节。
