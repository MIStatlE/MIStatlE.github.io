# 人工智能

这一部分关注人工智能领域的基础概念与主要方法，包括搜索策略、知识表示、规划、学习方法等。

## 基本概念

- 什么是人工智能，研究的目标和主要任务。
- 智能体模型与环境的交互。

## 搜索与规划

- **状态空间搜索**：深度优先搜索、广度优先搜索、A\* 算法等。
- **规划算法**：STRIPS、基于图的规划等。

## 知识表示与推理

- 逻辑表示、规则推理、贝叶斯网络等。
- 简述每种表示方法的特点与应用场景。

## 强化学习

- 主要概念：马尔可夫决策过程、价值函数、策略迭代等。
- 基本算法：Q 学习、SARSA 等。

## 其他主题

人工智能领域还包括自然语言处理、计算机视觉、多智能体系统等，可根据兴趣深入学习。
