# 机器学习 / Machine Learning

本节是关于机器学习理论与实践的学习笔记，涵盖监督学习、无监督学习和深度学习等主题。

=== "中文"

#### 监督学习

- **主要内容**：线性回归、逻辑回归、决策树、支持向量机等。
- **学习建议**：理解每种算法的假设、优点和局限，配合代码实践。

#### 无监督学习

- **主要内容**：聚类、降维、主成分分析等。
- **学习建议**：关注算法的适用场景，动手分析实际数据集。

#### 深度学习

- **主要内容**：人工神经网络、卷积神经网络、循环神经网络等。
- **学习建议**：学习基本网络结构与训练技巧，尝试构建简单模型。

#### 其他主题

例如半监督学习、迁移学习、强化学习等，可根据学习进度逐步填充。

=== "English"

#### Supervised Learning

- **Topics**: linear regression, logistic regression, decision trees, support vector machines, etc.
- **Study Tips**: Understand the assumptions, strengths and limitations of each algorithm and pair your study with code practice.

#### Unsupervised Learning

- **Topics**: clustering, dimensionality reduction, principal component analysis (PCA), etc.
- **Study Tips**: Pay attention to the scenarios where each algorithm is suitable and analyze real datasets hands‑on.

#### Deep Learning

- **Topics**: artificial neural networks, convolutional neural networks, recurrent neural networks and related architectures.
- **Study Tips**: Learn the basic network structures and training techniques, and try building simple models.

#### Other Topics

For example, semi-supervised learning, transfer learning and reinforcement learning. Fill in these sections gradually according to your learning progress.
