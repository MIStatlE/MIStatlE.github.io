- **Sorting Algorithms**: quick sort, merge sort, heap sort, etc.
- **Searching Algorithms**: binary search, hashing-based search, etc.
- **Graph Algorithms**: shortest path, minimum spanning tree, topological sort and more.
- **Dynamic Programming**: knapsack problem, longest common subsequence, etc.

#### Algorithm Analysis

- Definition of time and space complexity.
- Use of asymptotic notation (Big‑O, Big‑Omega, Big‑Theta).
- Comparison of common complexity classes and ideas for optimization.
