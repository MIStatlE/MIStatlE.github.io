# 数学基础

这一部分记录学习数学基础知识的笔记，包括微积分、线性代数、概率统计等内容。建议按主题逐步完善，每个小节用二级标题开始。

## 微积分

- 主要概念：极限、导数、积分等。
- 学习要点：掌握求导和积分的基本技巧以及相关定理。

## 线性代数

- 主要概念：向量与矩阵、线性变换、特征值与特征向量、矩阵分解等。
- 学习要点：理解矩阵运算及其几何意义，掌握常用分解方法。

## 概率与统计

- 主要概念：随机变量、概率分布、期望与方差、假设检验等。
- 学习要点：熟悉常见分布及其性质，能够应用基础统计方法分析数据。

记录数学基础中其他重要的主题，如优化方法、数值计算等。
