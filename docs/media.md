# 媒体内容

除了在网站上分享笔记和文章，我也会在其他平台发布视频和短内容，例如 B 站、小红书和知乎。本页用于汇总这些媒体资源，并示范如何在 Markdown 中嵌入或链接它们。

## B 站（Bilibili）

你可以在 B 站看到我分享的学习笔记、课程解读等视频。以下是一个示例嵌入方式，使用 `<iframe>` 标签将视频播放器插入页面：

```html
<iframe src="//player.bilibili.com/player.html?bvid=BV1XX411c7mD&page=1"
        scrolling="no" frameborder="no" framespacing="0" allowfullscreen>
</iframe>
```

请将示例中的 `bvid`、`page` 等参数替换为你自己的视频编号。

## 小红书

若你在小红书发布过课程笔记或短视频，可以通过普通链接形式放置在此页面。例如：

- [我的小红书主页](https://www.xiaohongshu.com/user/your-id)
- [示例笔记](https://www.xiaohongshu.com/explore/xxxxx)

由于小红书不提供官方嵌入代码，通常通过链接方式分享即可。

## 知乎

知乎文章或专栏可以直接使用 Markdown 链接引用。例如：

- [我的知乎文章：如何自学机器学习](https://zhuanlan.zhihu.com/p/123456789)

你也可以在此页面记录你在知乎发布的问答或专栏地址。

---

如果你还有其他平台（如抖音、微博等）的资源，也可以在此处按类似方式添加。通过集中整理媒体内容，读者可以方便地在不同平台之间跳转，获取更多形式的学习资料。

# Media content

Besides sharing notes and articles on this site, I also publish videos and short content on other platforms such as Bilibili, Xiaohongshu and Zhihu. This page is used to collect these media resources and demonstrate how to embed or link them in Markdown.

## Bilibili

You can see videos of my learning notes, course interpretations, etc., on Bilibili. Here's an example of embedding a video player using the `<iframe>` tag:

```html
<iframe src="//player.bilibili.com/player.html?bvid=BV1XX411c7mD&page=1"
        scrolling="no" frameborder="no" framespacing="0" allowfullscreen>
</iframe>
```

Please replace the `bvid`, `page` and other parameters in the example with your own video ID.

## Xiaohongshu

If you've posted course notes or short videos on Xiaohongshu, you can add links on this page, for example:

- [My Xiaohongshu home page](https://www.xiaohongshu.com/user/your-id)
- [Sample note](https://www.xiaohongshu.com/explore/xxxxx)

As Xiaohongshu doesn't provide official embed codes, it's usually shared via links.

## Zhihu

Zhihu articles or columns can be referenced using standard Markdown links, for example:

- [My Zhihu article: How to self‑learn machine learning](https://zhuanlan.zhihu.com/p/123456789)

You can also record the Q&A or column addresses you publish on Zhihu here.

---

If you have resources on other platforms (such as Douyin, Weibo, etc.), you can add them here in a similar manner. By organizing media content in one place, readers can easily switch between different platforms to access more forms of learning materials.
