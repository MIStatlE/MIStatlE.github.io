# 信息源

优质的信息源可以帮助你及时了解学科动态和最新研究成果。本页汇集一些常用的信息渠道。

## 在线平台

- **学术搜索引擎**：如 arXiv、Google Scholar，用于查找论文。
- **问答社区**：Stack Overflow、知乎等，便于交流和解决问题。
- **在线论坛**：V2EX、Reddit 的相关版块等。

## 新闻与博客

- 跟踪人工智能、机器学习领域的新动向的科技新闻网站。
- 优秀个人博客或团队博客，分享算法和实践经验。

## 会议与期刊

- 关注领域内顶级会议（如 NeurIPS、ICML、ACL 等）和期刊，获取最新研究成果。
- 提示如何查找会议论文、观看会议视频。

## 社区与组织

- 参与开源社区、加入学习小组，通过共同学习提高效率。
- 关注社交媒体上的学术账号，获取动态信息。
