Diffusion Models

diffusion
从前向加噪与反向去噪出发，系统学习 DDPM、DDIM、Flow Matching 与理论基础。

学习目标
从零到一理解扩散模型：动机 → 高斯扩散 → 反向 SDE/ODE → DDPM/DDIM → Flow Matching → 理论与实践。

扩散模型要解决的问题

生成模型希望学习未知分布 p^*(x)，从而生成近似样本。
直接从噪声“一步”生成复杂数据往往困难，而扩散模型的关键在于：

  1. 前向扩散（Forward Diffusion）：逐步加噪，使数据分布趋近高斯；
  2. 反向过程（Reverse Diffusion）：学习去噪映射，把噪声还原为数据；
  3. 每一步都只需建模相邻分布 p(x_{t-\Delta t}\mid x_t)，学习更稳定。

核心等价关系：
\text{Denoising} \;\Leftrightarrow\; \text{Learning } \nabla_x \log p_t(x)
学习去噪即学习“得分函数”，方向即密度上升最快的方向。

模型概览

模型 核心特征 适合理解的方向
DDPM (Ho et al., 2020) 随机反演，采样多样性强 适合学习随机微分方程与噪声调度思想
DDIM (Song et al., 2020) 确定性采样，速度快 适合工程与推理效率研究
Flow Matching (Lipman et al., 2023) 统一连续流框架 适合理解 ODE/SDE 统一与几何流
