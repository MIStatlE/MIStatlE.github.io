1. Why Diffusion?

给定来自未知分布 (p^*(x)) 的 i.i.d. 样本（如狗的图片），目标是构造一个采样器再生出近似的样本。
扩散模型的“巧思”在于：把“难的一步到位生成”改写成前向加噪（复杂 () 简单）与反向去噪（简单 () 复杂）的多步可逆过程，每一步都更容易学习。

Note关键视角
  • 前向过程（Forward）：(x_{t+t} = x_t + _t, _tN(0,_q^2t))
  • 反向过程（Reverse）：学习 (p(x_{t-t}x_t)) 或其均值（回归/得分）
  • 得分函数：(_x p_t(x)) 连接去噪与生成（Tweedie/Score Matching）

3. Core Concepts（核心概念）

3.1 Forward Diffusion（前向加噪）

[ x_{t+t} = x_t + _t,_tN(0,_q^2t), ] 使 (p_t) 逐渐趋近高斯；离散步数 (T) 越大，相邻分布越“近”。

3.2 Reverse SDE / DDPM（随机反演）

通过反向 SDE 采样：在每一步对 (x_t) 注入小随机性，保证分布覆盖与多样性。

3.3 Probability Flow ODE / DDIM（确定性反演）

用概率流方程得到确定性轨迹：相同 ({p_t}) 边缘分布下，轨迹唯一、可重复，适合加速与编辑。

3.4 Tweedie 与得分

在高斯观测下， [ (x)=E[x] = x + ^2_xp(x), ] 说明最优去噪方向就是密度上升最快方向；学习去噪()学习得分。

4. Prerequisites（先修清单）

  • 概率论与统计：条件期望、KL/JS、集中不等式（选修）
  • 随机过程：布朗运动、伊藤引理、Fokker–Planck（可边学边用）
  • 优化/深度学习：MSE/L2，AdamW，学习率与噪声调度
Tip

建议边学边做：在玩具 2D 数据上可视化前向/反向轨迹，直观看到“加噪→降噪”的几何过程。

5. Study Plan（8–10 周）

  • Week 1–2：复现最小去噪器（单步/多步），理解 MSE 与 (E[x_{t-t}x_t])
  • Week 3–4：实现 DDPM 采样；分析噪声调度、步数与 FID/IS
  • Week 5–6：切到 DDIM（ODE）；做步数扫描（50→20→10）与一致性对比
  • Week 7–8：读 Flow Matching/Consistency，做一次蒸馏或单步模型尝试
  • Week 9+：专项：编辑/插值/视频/控制，或做一篇读书/实验报告

6. Assignments（实践作业）

  1. Toy-2D 可视化：画出 ((p_{t-1},p_t)) 以及反向均值轨迹
  2. 回归等式验证：数值验证 (f E|f(x_t)-x{t-1}|^2 = E[x_{t-1}x_t])
  3. DDPM↔︎DDIM 对照：相同步数下比较质量、多样性与速度
  4. 得分一致性：估计 (_xp_t) 并验证 Tweedie 方向的“拉回”效应

8. FAQ（常见困惑）

  • DDPM vs DDIM：谁更好？
    DDPM 多样性更强、理论更贴近 SDE；DDIM 轨迹确定、采样快、适合编辑。实际常两者结合：先用 DDIM 快速预览，再用 DDPM 精修。
  • 为什么学习“均值”就够了？
    小步噪声下 (p(x_{t-t}!x_t)N(,^2 I))，已知方差，只需学均值（做回归）即可近似整个条件。
  • Tweedie 与扩散的关系？
    都把“去噪方向”与“得分 (p)”绑定：扩散的反向均值沿着 Tweedie 方向修正。
